About

Pets In A Row is a version of the classic tic-tac-toe board game. It is played by two opponents (player and computer), who take turns marking the squares on a game board. The object of the game is to place three successive marks in a horizontal, vertical, or diagonal row on the grid. The opponents alternate moves, with the player touching the desired square in the grid to place a mark, followed by the computer marking a square. The opponent who first obtains three-in-a-row wins; if neither opponent is able to obtain three-in-a-row and all moves are exhausted the game is declared a draw.

Game Play

At startup Pets In A Row displays the game board on the playing screen. If the player is configured to make the first move, he touches the desired square on the board to place a mark in the square. The computer then responds by marking one of the board squares. (Note that if the computer is configured to make the first move it marks one of the board squares when the New Game button is selected).
At the conclusion of a game the playing screen displays the outcome (win, loss, or draw). A new game will automatically start shortly after the outcome is displayed. You can also quit a game in progress or start a new game by touching the New Game button.
To view/update game statistics, configure game settings, access Game Center, or view support info, the player accesses the game menu by touching the Info icon at the top of the playing screen.

Game Interface

The game interface includes a playing screen and several additional screens used to perform game configuration and review game stats.
Playing screen: Consists of the game board, an Info icon (click to display the menu screen), a game status field (displays game win, lose, draw status), and a New Game button (touch to start a new game).
Menu screen: Resume an in-progress game, start a new game, select the game settings screen, select the statistics screen, select the support screen, or access Game Center.
Settings screen: turn on/off sound effects, enable/disable Game Center, select the appropriate pet (cat or dog) for the player, select which opponent (player or computer) makes the first game move, select game level-of-difficulty when the player makes the first move (easy, medium, hard), start/resume a game, display the menu screen (Info icon).
Statistics screen: View and reset win/loss/draw statistics for player, start/resume a game, display the menu screen (Info icon).
Support screen: View this support info, display the menu screen (Info icon).
Game Center screen: Access Game Center, where the player can review achievements and/or issue challenges to friends.

Game Settings

Sound Effects: Enable/disable sound effects played on each move and when a game is completed.
Game Center: Enable/disable game center support for this app.
First Move: Select Player or Computer to configure which opponent makes the first move in a game. If the computer is selected to make the first move, the level of difficulty for the game is automatically set to Expert.
Player Pet: Select the pet (cat or dog) that is used to mark the player's squares on the game board; the computer then uses the other pet to mark its squares.
Level of Difficulty: The game has three levels of difficulty when the player makes the first move (easy/medium/hard), and a fourth (expert) level that is set when the computer makes the first move. Use this control to select the desired level for a game when the player makes the first move. This selection will automatically start a new game.

Statistics

The statistics screen displays the number of wins, losses, and draws for the player at each level-of-difficulty. The statistics for a level can be reset by touching the corresponding X (clear stats) button.

Game Center

Game Center is the Apple social gaming network. From this app you can view your current achievements for 3 In A Row! and issue challenges to friends.
